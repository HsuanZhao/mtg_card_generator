# 🃏 万智牌卡片生成器 Pro — 法律版

> 一款专为《万智牌》爱好者设计的**卡片生成与定制工具**，采用纯前端技术，无需服务器，所有数据均在本地处理。  
> 本版本以 **“符文石板”** 为主题，打造古朴、厚重的石质浮雕风格，支持高度自定义：文字、颜色、插画、阵营图标、字体、字号，并一键导出高清 PNG。

---

## ✨ 主要特性

- 🪨 **独特的石板风格** — 双层雕刻边框、凹凸石纹纹理、仿凿刻的栏目条，模拟真实石碑质感。
- 🎨 **丰富的颜色预设** — 内置寒霜蓝、黑曜岩、赤焰岩、苔藓绿等 7 套配色，一键切换，也可手动调色。
- 🧩 **灵活的内容编辑** — 名称、类型、规则文本、风味文字、力量/防御、费用数字、底部编号，全部可实时修改。
- 🖼️ **自定义插画** — 上传本地图片，自动裁剪填充至卡片画框，支持横版推荐尺寸。
- ⚔️ **阵营图标系统** — 内置「民事」「刑事」「行政」「执行」四种 SVG 符号，可置于类别栏右侧。
- 📐 **字体 & 字号调整** — 支持多款中英文字体（楷体、宋体、黑体、Beleren 等），所有字号独立滑块调节。
- 💾 **一键导出 PNG** — 使用 `html2canvas` 渲染为高清图片（3 倍缩放），保存到本地。
- ♻️ **石纹随机再生** — 每次点击可重新生成噪点纹理，使每张卡片的背景肌理独一无二。

## 🚀 快速开始

### 方法一：直接使用（无需安装）
1. 将 `index.html` 保存到本地。
2. 用现代浏览器（Chrome / Edge / Firefox）打开该文件。
3. 所有功能即刻生效，离线可用（仅需外网加载 `html2canvas` CDN，或下载后替换为本地引用）。

## 🧭 使用指南

### 左侧控制面板（按组划分）

| 分组 | 说明 |
|------|------|
| **基础信息** | 卡牌名称、类别行、力量/防御、费用数字（显示在费用符石内） |
| **规则文本** | 正文描述（支持换行）和风味文字（斜体） |
| **阵营图标** | 单击选择一个图标，类别栏右侧会显示对应符号 |
| **中央插画** | 点击按钮上传图片，自动填充到画框区域 |
| **字体 & 字号** | 分别选择标题/正文字体，并拖动滑块调整各部分的字号 |
| **颜色风格** | 点击预设色块快速切换，也可展开下方色值手动微调 |
| **底部信息** | 自定义左下角编号/稀有度/版本标记 |
| **导出** | 点击按钮生成 PNG 并自动下载（文件名为卡牌名称） |

### 实时预览
右侧预览区会随任何输入、选择或滑动而**即时更新**，所见即所得。

### 石纹再生
每次点击「🪨 重新生成石纹肌理」按钮，都会为卡片背景的颗粒噪点与光照凹凸赋予新的随机种子，使纹理不重复。

## 🎨 自定义技巧

- **插画建议**：推荐横版图片，宽高比约 **1.76:1**（如 1300×740、1760×1000），主体居中偏上，避免裁剪时丢失关键元素。
- **颜色搭配**：可通过“石板主色 A/B”和“辅色”调节背景基调，“栏目色”控制标题栏与类型栏的饰面色。
- **字体选择**：若系统未安装 Beleren 字体，将自动回退到衬线体，不影响中文显示；英文/数字字形可选用 Georgia 等衬线字体提升质感。

## 🛠️ 技术实现

- **核心语言**：HTML5 + CSS3 + 原生 JavaScript (ES6)
- **渲染引擎**：`html2canvas` (v1.4.1) – 将 DOM 节点转为 Canvas 并导出图片
- **图形特效**：利用 SVG `feTurbulence` 和 `feDiffuseLighting` 滤镜生成石质纹理与光照凹凸
- **样式架构**：CSS 变量（Custom Properties）实现动态换色与字号调节
- **完全前端**：所有数据处理均在浏览器内完成，无服务器交互，保护隐私
